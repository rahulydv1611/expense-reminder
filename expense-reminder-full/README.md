Expense Reminder Full Project (Node.js backend + React frontend)
Files:
- backend/: Express server, DB schema, env example
- frontend/: Vite + React app, env example
How to use:
1) Install Node.js and npm
2) Setup Postgres (Supabase recommended) and run backend/db.sql
3) Configure backend/.env and frontend/.env
4) Start backend and frontend locally, then deploy using Render/Netlify
